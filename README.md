# Speech-Based-Ayurvedic-Prakriti-Prediction
Speech-Based Ayurvedic Prakriti Prediction is an academic research project that uses speech signal analysis and machine learning to classify Vata, Pitta, and Kapha Prakriti types. The study was conducted at KLE Technological University, Dr. M.S. Sheshgiri College of Engineering and Technology, in collaboration with IIIT Dharwad.
